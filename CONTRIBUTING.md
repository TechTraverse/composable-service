# Contributing

Pull requests and issues are welcome. Contributors are expected to adhere to
the Code of Conduct adhere the contribution guidelines below.

## Pull requests

When creating a pull request for this repository, please format commit messages
in the format described in the [Git Commit
Guidelines](https://git-scm.com/book/en/v2/Distributed-Git-Contributing-to-a-Project#_commit_guidelines)
and in [Tim Pope's
blog](https://tbaggery.com/2008/04/19/a-note-about-git-commit-messages.html).
To the extent it is applicable, please format the Pull Request title and
description in the same way.
